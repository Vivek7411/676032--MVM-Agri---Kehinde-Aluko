# AgriMarket Nigeria — Bulk Agricultural Marketplace

A wireframe prototype for Nigeria's bulk agricultural marketplace. All products are sold by weight (kg or metric tonne) only.

## Quick Start

```bash
bun install
bun run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set Source to **GitHub Actions**
4. Push to `main` branch — the workflow in `.github/workflows/deploy.yml` will auto-deploy
5. Your site will be live at `https://YOUR_USERNAME.github.io/agrimarket-nigeria/`

## Tech Stack

- Next.js 16 (App Router)
- TypeScript
- Tailwind CSS 4
- shadcn/ui components
- Zustand state management
